# A股交易分析工具

一个专为 iPhone 设计的 A 股交易分析工具，支持**底仓独立管理**、**先进先出交易配对**、**动态成本均价**、**一键更新股价**和**日历收益视图**。

## 功能特点

- ✅ 底仓与交易完全独立，底仓永不参与配对
- ✅ 动态成本均价：`(累计买入金额 - 累计卖出金额) / 净股数`
- ✅ 浮动盈亏：`净股数 × (现价 - 成本均价)`
- ✅ 先进先出配对，自动计算已实现盈亏
- ✅ 一键更新所有持仓股票最新价（东方财富接口）
- ✅ 三个标签页：持仓（合并底仓+交易净头寸）、交易统计、交易流水
- ✅ 日历收益视图：每日、每月、每年已实现盈亏，点击日期查看明细
- ✅ 费率可调（默认佣金 0.01345%，印花税 0.05%）

## 如何使用

1. 下载 `index.html` 文件
2. 用浏览器（Safari/Chrome）打开
3. 点击右上角 ⚙️ 设置底仓和费率
4. 添加交易记录，系统自动匹配并计算盈亏
5. 点击“打开日历收益视图”查看历史收益
